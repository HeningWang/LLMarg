# Meeting Notes — 2026-03-20

## Goal

We will start with a grid search for the listener-side experiment.

The initial search space will vary along four dimensions:

1. Probing method
2. Temperature
3. Model family
4. Model size

The probing-method variants should be informed by the paper "LLMprobing" in this folder and the related paper at https://arxiv.org/pdf/2403.00998.

## Grid Search Axes

### 1. Probing Methods

We will first compare several prompt / probing setups for the listener task.

- Direct label prediction: choose one of `high`, `low`, `info`
- Role-description prompting: explicitly describe speaker goals before prediction
- Chain-of-thought style probing: ask for short reasoning before final label
- Structured justification probing: require evidence fields before final choice
- Calibration variant: ask for label plus confidence score
- Log probabilities: extract distributional information directly from the model

Note: the exact probing variants should be refined after extracting the main alternatives from the LLM probing paper.

### 2. Temperature

We will test three temperature settings:

- Low
- Mid
- High

Proposed default mapping if we want concrete values immediately:

- Low = 0.2
- Mid = 0.7
- High = 1.2

### 3. Model Families

We will start with several Hugging Face model families.

Candidate starting set:

- Llama family
- Mistral family
- Qwen family
- Gemma family

### 4. Model Sizes

Within each family, we will compare three size bands:

- Low
- Mid
- High

Working interpretation:

- Low: small instruct model available on local / affordable inference
- Mid: medium model with feasible batch evaluation
- High: largest instruct model we can reliably run for the first pass

Exact model names should be finalized after checking availability, context length, and inference cost.

## Recommended Execution Order

To control complexity, we should not run the full grid immediately.

Phase 1:

- Fix one model family and one model size
- Sweep probing methods and temperature only
- Identify 1 to 2 probing setups that are stable and interpretable

Phase 2:

- Hold the best probing setup fixed
- Expand to multiple model families and size bands
- Run the broader comparison

Phase 3:

- Run analysis across all completed conditions
- Decide whether a second-round targeted grid search is needed
